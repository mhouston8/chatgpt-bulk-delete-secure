Privacy Policy

Last updated: May 25, 2024

Overview

ChatGPT Bulk Delete Secure is committed to protecting your privacy. This extension is designed to help you manage your ChatGPT conversations by providing bulk deletion functionality.

Data Collection and Usage

This extension:

  • Does not collect any personal information
  • Does not store any data
  • Does not transmit any data to external servers
  • Only operates on the ChatGPT interface when explicitly activated by the user

Permissions Used

The extension requires the following permissions:

  • activeTab: Used to access the ChatGPT interface only when the extension is activated
  • scripting: Used to perform the bulk deletion functionality
  • Host permission for chat.openai.com: Required to access the ChatGPT interface

Data Security

All operations are performed locally in your browser. The extension does not have access to your ChatGPT account credentials or any other sensitive information.

Contact

If you have any questions about this privacy policy, please contact us through the developer contact information provided in the Chrome Web Store listing.
